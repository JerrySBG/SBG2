### INSTALL SCRIPT 
```
apt install -y && apt update -y && apt upgrade -y && wget -q https://raw.githubusercontent.com/JerrySBG/SBG2/main/premi.sh && chmod +x premi.sh && ./premi.sh
```

## UPDATE SCRIPT
```
wget -q https://raw.githubusercontent.com/JerrySBG/SBG2/main/update.sh && chmod +x update.sh && ./update.sh
```
